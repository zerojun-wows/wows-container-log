# wows-container-log

Tool to log your container drops in World Of Warships.

## Dependencies

- Python 3.13
- PySide6
- SQLModel

## Usage

While in development you need to run

```bash
uv run wows-container-log
```

in project folder

## Project folder structure

```text
wows-container-log
├─ doc
├─ src
│  └─ wows_container_log
│     ├─ gui
│     ├─ models
│     └─ storage
└─ tests
```
